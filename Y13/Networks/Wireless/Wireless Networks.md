
### What is a 'Hot Spot'

Any public WiFi access users can freely connect to


## The purpose of WiFi

WiFi is a wireless networking technology providing high-speed internet and network connections
Devices connect via the internet via a Wireless network Access Point (WAP)
WiFi 'hot spot' can often be found in:
	- Cafes
	- Hotels
	- Libraries
	- Other public places<br>


### Connecting to a Wireless network

Whats needed?
	- ISP (Internet service provider)
	- Modem and wireless router (often combined)
	- Device with a network interface card (NIC), also known network interface controller<br>

### Wireless components

- Wireless Network Interface Card (NIC)
- A station consists of a computer and NIC
- Stations share a radio frequency channel
- Wireless access point requires a connection to a router, and the router requires a connection to a modem
	- The WAP and the modem are often built into the router<br>

### Securing a wireless network

- Use a password
- Protect your SSID
- Encrypt the signals (WPA)
- Restrict access
	- MAC Filtering by blacklisting and whitelisting
- Hardware firewall
- Stop SSID broadcast
	- This makes it so that people dont even know the network is there


### Service Set Identifier

Identifies each network by a unique name
- Must be used by all devices on that network
- Can be set manually or automatically
- can be hidden to make it harder to detect
- Can be set to broadcast to wireless devices in range of an access point
