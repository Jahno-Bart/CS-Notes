## Golden Rule

If you have X bits then you can represent $2^x$ possible different values

They range from 0 to $(2^x - 1)$

For example: if you have 8 bits, you can have $2^8$ possible values with a range of 0 - 255 

(0000 0000 --> 1111 1111)
